A idle game project in development.

# Front-end instructions

#### Prerequisites

* NodeJS

Assuming that you've rubygems installed.
```
gem install bundle
bundle install
```

Then install the `nodejs` dependency:
```
npm install
```

## Grunt

1. `grunt`
2. enjoy
